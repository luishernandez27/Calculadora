# Calculadora

Aplicación Android desarrollada en Java con Android Studio.

## Funcionalidades
- Suma
- Resta
- Multiplicación
- División
- Limpiar campos
- Salir de la aplicación

## Componentes utilizados
- TextView
- EditText
- Button
- ImageView
- LinearLayout
- OnClickListener
- Toast
